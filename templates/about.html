Project Overview

Welcome to our Disease Prediction and Health Advice application. This project aims to assist users in identifying potential diseases based on their symptoms and provide related health advice, including precautions, medications, diets, and workouts.

Technology Stack

We use a combination of cutting-edge technologies to build and deploy this application:

  • Python & Flask for the backend
  • scikit-learn for machine learning
  • Pandas and NumPy for data manipulation
  • HTML, CSS, and JavaScript for the frontend
  • Bootstrap for responsive design
