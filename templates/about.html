About HealthBuddy

Project Overview

HealthBuddy is a web-based application designed to assist users in predicting potential diseases based on symptoms they provide. The application offers additional healthcare information, including disease descriptions, precautions, medications, recommended diet, and workout suggestions. It also features a chatbot for interactive assistance.

Technology Stack

We use a combination of cutting-edge technologies to build and deploy this application:

  • Python & Flask for the backend
  • scikit-learn for machine learning
  • Pandas and NumPy for data manipulation
  • Pandas and NumPy for data manipulation
  • HTML, CSS, and JavaScript for the frontend
  • Google GenerativeAI (Gemini API) for Chatbot Integration
  • Bootstrap for responsive design
